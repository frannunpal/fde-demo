# FDE-TEMPLATE

Hello! And welcome to the codesandbox demo (and template) for fde-desktop!

I'm Fran, and am very pleased that you took the time to take a look at this demo, and sincerely hope you have a good time taking a look at how the project is coded and the examples provided with this demo. For now it replicates the same environment that [my github pages](https://frannunpal.github.io/) At the time of writing, codesandbox is using [fde-core:0.4.22](https://www.npmjs.com/package/@fde-desktop/fde-core). I keep adding more functionality to the package at least once a week.

You have everything you need to create a new app in here, docs, tests, etc. However keep in mind that most of the docs are llm updated, so they might not be 100% up to date (albeit I try to update them reguarily)

You can access these docs from codesandbox or in the integrated text editor in fde-desktop.

If you find any issues, you can [report them in the public repo](https://github.com/frannunpal/frannunpal.github.io/issues)(you know the drill)

Have fun and [contact me](mailto:frannunpal@gmail.com) with anything you find in this demo! Ill be pleased to help you with watever you need
